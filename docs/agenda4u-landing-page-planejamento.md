## 1. Estrategia Geral

### Objetivo unico
Converter visitantes em trials gratuitos (14 dias, sem cartao).

### Posicionamento
"O sistema de gestao para profissionais da saude com IA inclusa. A partir de R$ 29,90/mes."

### Tom
Direto, profissional, sem ser frio. O profissional de saude esta cansado de sistemas caros e complicados. Nos falamos a lingua dele.

### Rota no projeto
`/` (rota raiz) — quando logado, redireciona para `/dashboard`. Quando nao logado, mostra a landing page.

## 3. Elementos de Design

### Cores (do design system existente)
- Primary: Teal `#0D9488` (botoes, links, destaques)
- Primary Dark: `#115E59` (hover, titulos fortes)
- Background: `#F8FAFC` (Slate 50)
- Cards: `#FFFFFF`
- Texto principal: `#0F172A` (Slate 900)
- Texto secundario: `#64748B` (Slate 500)
- Accent: `#F97316` (Orange — badge de destaque, "Mais popular")

### Tipografia
- Fonte: Inter (Google Fonts)
- Hero headline: 36-48px, peso 600
- Secao headlines: 28-32px, peso 600
- Body: 16-18px, peso 400
- CTA buttons: 16px, peso 600

### Layout
- Max-width do conteudo: 1200px, centralizado
- Mobile-first: coluna unica ate 768px
- Desktop: grid 2-3 colunas conforme secao
- Espacamento entre secoes: 80-120px

### Componentes reutilizaveis
- `HeroSection`
- `PainPointCards`
- `FeatureGrid`
- `AIHighlight`

- `ComparisonTable`
- `SpecialtyGrid`
